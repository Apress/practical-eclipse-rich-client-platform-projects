Help System Benefits

Eclipse provides a powerful help system that can be used to add documentation to RCP quickly with minimal effort. The help system provides the following benefits to RCP:

* It uses a browser-based presentation and therefore has full HTML support.

* It is supported on all of the Eclipse platforms.

* It provides powerful built in search.

* It provides localization for additional languages.

* It supports features such as context help and keyword index provided the appropriate content files are used.

* It supports a web based InfoCenter to host the documentation on the internet.

* It provides programmatic extension points for content producers to generate dynamic help including table of contents, keyword index, and content. This can be useful for document format conversion.

* It is the easiest and fastest way to add that documentation support to RCP.

The help system is a robust, feature rich and extensible component, which continues to grow in terms of quality and features.
